## 💾 Abra o projeto localmente

Clone o projeto

```bash
  git clone https://github.com/leodsant/portfolio.git
```

Vá para o diretório do projeto

```bash
  cd portfolio
```

Instale as dependências

```bash
  npm install
```

Starte o server

```bash
  npm run dev
```
